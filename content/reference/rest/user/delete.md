Deletes a user by id.


# Method

DELETE `/user/{id}`


# Parameters

## Path Parameters

<table class="table table-striped">
  <tr>
    <th>Name</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>id</td>
    <td>The id of the user to be deleted.</td>
  </tr>
</table>


# Result

This method returns no content.
