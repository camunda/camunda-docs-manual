Unlocks a user by id.


# Method

POST `/user/{id}/unlock`


# Parameters

## Path Parameters

<table class="table table-striped">
  <tr>
    <th>Name</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>id</td>
    <td>The id of the user to be unlocked.</td>
  </tr>
</table>


# Result

This method returns no content.
