Removes a member from a group.


# Method

DELETE `/group/{id}/members/{userId}`


# Parameters

## Path Parameters

<table class="table table-striped">
  <tr>
    <th>Name</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>id</td>
    <td>The id of the group.</td>
  </tr>
  <tr>
    <td>userId</td>
    <td>The id of user to remove from the group.</td>
  </tr>
</table>


# Result

This method returns no content.
