Adds a member to a group.

# Method

PUT `/group/{id}/members/{userId}`

# Parameters

## Path Parameters

<table class="table table-striped">
  <tr>
    <th>Name</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>id</td>
    <td>The id of the group.</td>
  </tr>
  <tr>
    <td>userId</td>
    <td>The id of user to add to the group.</td>
  </tr>
</table>



# Result

This method returns no content.
