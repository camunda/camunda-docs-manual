Deletes a membership between a tenant and an user.

# Method

DELETE `/tenant/{id}/user-members/{userId}`

# Parameters

## Path Parameters

<table class="table table-striped">
  <tr>
    <th>Name</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>id</td>
    <td>The id of the tenant.</td>
  </tr>
  <tr>
    <td>userId</td>
    <td>The id of the user.</td>
  </tr>
</table>

This method returns no content.
