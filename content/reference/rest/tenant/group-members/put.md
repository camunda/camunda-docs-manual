Creates a membership between a tenant and a group.

# Method

PUT `/tenant/{id}/group-members/{groupId}`

# Parameters

## Path Parameters

<table class="table table-striped">
  <tr>
    <th>Name</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>id</td>
    <td>The id of the tenant.</td>
  </tr>
  <tr>
    <td>groupId</td>
    <td>The id of the group.</td>
  </tr>
</table>
