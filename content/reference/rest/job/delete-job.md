Deletes a job by id.


# Method

DELETE `/job/{id}`


# Parameters

## Path Parameters

<table class="table table-striped">
  <tr>
    <th>Name</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>id</td>
    <td>The id of the job to be deleted.</td>
  </tr>
</table>


# Result

This method returns no content.

    <td></td>
    <td>Request successful.</td>
  </tr>

# Example

## Request

DELETE `/job/aJobId`

## Response

Status 204. No content.
